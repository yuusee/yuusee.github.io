# `Windows`下的问题
在`Windows`下，大部分情况我们都是通过鼠标来找到程序打开，
那有没有像 `Linux` 一样输入命令可以让程序运行的解决方案呢？


# 解决
其实在可以通过环境变量的方式，将自己常用程序的快捷放到一个指定的文件夹内，
使用时通过 `Win+R`， 输入常用程序的快捷方式，回车即可运行。
这样是不是瞬间提高了一个 `BG`~
